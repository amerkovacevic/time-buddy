# Time Bro

A modern, beautiful time zone converter app built with React, Vite, and Tailwind CSS. Easily convert time zones with a smooth sliding interface.

## Features

- **Dynamic Time Zones**: Add as many time zones as you need
- **Smooth Slider**: Navigate through time zones with a beautiful horizontal slider
- **Real-time Updates**: All times update every second
- **Modern UI**: Built with universal branding and Tailwind CSS
- **Search & Filter**: Quickly find time zones from a comprehensive list
- **Custom Labels**: Give your time zones friendly names

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Usage

1. The app starts with your local time zone
2. Click the "+" button to add more time zones
3. Use the arrow buttons or swipe to navigate between time zones
4. Hover over a time zone card to see the remove button
5. Search for specific cities or browse popular/common time zones

## Technology Stack

- **React 18**: UI framework
- **Vite**: Build tool and dev server
- **Tailwind CSS**: Styling with universal design tokens
- **Intl API**: Native browser time zone support

## Universal Branding

This app follows the universal branding system defined in `shared-design-tokens.js`, ensuring consistency across all applications.
